# Seijinsiki.01

# 掛川北中学校 同窓会案内サイト

Instagramのプロフィールリンクから誘導するための、**モバイルファーストな1ページWebサイト**です。  
掛川北中学校の同窓会について、開催概要・出欠回答・写真共有・クラスごとの情報共有・幹事への連絡導線を、スマホで見やすくシンプルにまとめています。

---

## 概要

このサイトは、同窓会参加者がスマホから迷わず必要な情報にアクセスできるように設計されています。

### 主な目的
- 同窓会の開催案内を見せる
- 出欠確認フォームへ誘導する
- 写真共有アルバムへ誘導する
- クラスごとの情報共有リンクを案内する
- 幹事への連絡導線を用意する
- 参加予定の先生リストを掲載する

## 対応している機能

### 1. ヒーローセクション
- タイトル表示
- 同窓会の概要説明
- 出欠回答CTA
- 写真共有CTA
- 幹事連絡CTA

### 2. 開催概要
以下をカード形式で表示しています。
- 学校名
- 開催日
- 時間
- 会場
- 会費
- 回答締切

### 3. クイックメニュー
スマホで押しやすい導線として、以下のリンクを上部に配置しています。
- 出欠回答
- 写真共有
- 幹事へ連絡

### 4. クラスごとの情報共有
以下のクラスごとに専用リンクを設定できます。
- 3年1組
- 3年2組
- 3年3組
- 3年4組
- 3年5組

### 5. 写真共有
Googleフォトなどの共有アルバムURLにリンクできます。

### 6. 参加予定の先生
先生の名前を一覧形式で掲載できます。

### 7. よくある質問
最低限のFAQを掲載しています。
- 途中参加はできますか？
- 欠席でも連絡した方がいいですか？
- 写真は誰でも見られますか？
- クラスごとの連絡はどこで見られますか？

### 8. お問い合わせ
Instagram DMへの導線を用意しています。

## デザイン方針

- **モバイルファースト**
- Instagramから来た人が**3タップ以内で目的に到達**できる構成
- 明るく上品で、成人式・同窓会らしいやわらかな雰囲気
- 過度にビジネスっぽくしない
- 同級生向けに親しみやすいUI
- シンプルで迷わない設計
- 白〜淡いクリーム系の背景
- ネイビー / ゴールド系の落ち着いたアクセントカラー
- 丸みのあるカードUI
- 大きめのボタンでタップしやすい設計

---

## 技術構成

- **React**
- **Tailwind CSS**
- **shadcn/ui**
- **lucide-react**
- **framer-motion**

### 特徴
- 1ページ構成
- レスポンシブ対応
- モバイルファースト
- バックエンド不要
- ログイン不要
- 外部URLへ遷移するだけのシンプルな構成
- URL定数をまとめているため保守しやすい

---
